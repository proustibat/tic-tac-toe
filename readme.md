# Tic Tac Toe 
A simple tic tac toe game in ES6

# Installation

```
 git clone git@github.com:proustibat/tic-tac-toe.git
 cd tic-tac-toe.git
 yarn install
```
*You can use `npm install` if you would rather not use yarn.*

# Watching for development
```
yarn start

// or 
npm start
```

# Building
```
yarn build

// or 
npm run build
```
